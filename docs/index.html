Como jogar o jogo

1 - Primeiramente clique no botão Jogar a baixo.

2 - Após acessar o link, clique no botão verde e clique em "Download ZIP" para baixar o pasta compactada com os arquivos do jogo.

3 - Após efetuar o download da pasta, abra ela no seu IDE de prefêrencia.

4 - Assim que todos os arquivos estiverem prontos, execute o arquivo "jogo.py".

5 - Assim que a tela do jogo abrir, tudo estará pronto para jogar.
